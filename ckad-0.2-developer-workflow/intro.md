# Developer Workflow

In this lab you will practice the Kubernetes developer workflow: generating YAML, deploying apps, iterating with edits, and applying declarative updates.

These patterns form the core of how you will work through CKAD exam tasks — generate a baseline with imperative commands, then modify YAML to meet specific requirements.

**What you will learn:**
- Generate YAML with `--dry-run=client -o yaml`
- Edit and re-apply manifests
- Use `kubectl diff` to preview changes

**Time:** 30 minutes

**Curriculum module:** [CKAD Part 0 - Developer Workflow](https://kube-dojo.github.io/k8s/ckad/part0-environment/module-0.2-developer-workflow/)

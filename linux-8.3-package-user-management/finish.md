# Congratulations!

You have completed the Package and User Management lab.

## What You Learned

- Installing packages and holding versions with `apt-mark`
- Automated user creation with group membership
- System cleanup and maintenance routines

## Key Takeaways

1. **Hold critical packages** in production to prevent surprise upgrades
2. **Automate user management** — never create users manually one by one
3. **Regular maintenance** (cache cleanup, updates) prevents drift and saves disk
4. **apt-get clean** can recover significant disk space on busy systems

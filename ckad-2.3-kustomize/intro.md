# Kustomize

In this lab you will use Kustomize — built into kubectl — to manage configuration overlays without templates. Kustomize is on the CKAD exam.

**What you will learn:**
- Create base kustomizations with resources
- Build overlays that patch base resources
- Apply kustomize output with `kubectl apply -k`

**Time:** 30 minutes

**Curriculum module:** [CKAD Part 2 - Kustomize](https://kube-dojo.github.io/k8s/ckad/part2-deployment/module-2.3-kustomize/)

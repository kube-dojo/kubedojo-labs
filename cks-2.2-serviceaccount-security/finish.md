# Congratulations!

You have completed the **ServiceAccount Security** lab.

## Key Takeaways

- Disable automatic token mounting for pods that do not need API access
- Use projected tokens with short expiration for API-accessing pods
- Audit ServiceAccount permissions regularly
- Each workload should use its own dedicated ServiceAccount

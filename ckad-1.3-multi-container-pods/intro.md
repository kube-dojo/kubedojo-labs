# Multi-Container Pods

In this lab you will create pods with multiple containers sharing volumes and network. Multi-container patterns (sidecar, init, adapter) are heavily tested on the CKAD.

**What you will learn:**
- Init containers that run before app containers
- Sidecar containers that share a volume with the main app
- Adapter containers that transform data

**Time:** 35 minutes

**Curriculum module:** [CKAD Part 1 - Multi-Container Pods](https://kube-dojo.github.io/k8s/ckad/part1-design-build/module-1.3-multi-container-pods/)

# Congratulations!

You have completed the **Container Image Security** lab.

## Key Takeaways

- Pin images by digest for immutable, verifiable deployments
- Restrict allowed registries with admission control
- Use minimal base images (distroless, alpine) to reduce CVE surface
- Multi-stage builds keep production images clean

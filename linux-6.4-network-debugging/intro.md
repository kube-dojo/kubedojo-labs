# Network Debugging

Network issues are among the most common problems in production systems. This lab teaches you a systematic approach to network debugging using standard Linux tools.

## What You Will Learn

- Connectivity testing with `ping` and `traceroute`
- DNS debugging with `dig` and `nslookup`
- Port scanning with `nmap` and `nc`
- Packet capture with `tcpdump`

## Prerequisites

- Basic TCP/IP networking concepts
- Understanding of DNS, ports, and protocols

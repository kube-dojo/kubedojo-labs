# Congratulations!

You have completed the Network Debugging lab.

## What You Learned

- Connectivity testing with `ping` and `traceroute`
- DNS debugging with `dig`
- Port scanning with `nmap`
- Packet capture with `tcpdump`

## Key Takeaways

1. **Start simple** — ping first, then work up to tcpdump
2. **DNS is usually the culprit** — always check DNS resolution early
3. **ss -tlnp** quickly shows what is listening where
4. **tcpdump is non-negotiable** for deep network debugging — learn its filters

## Next Steps

- Learn Wireshark for graphical packet analysis
- Study eBPF network tracing for advanced debugging

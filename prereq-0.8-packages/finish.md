## Lab Complete!

### What You Learned

- **Update packages**: `apt update` refreshes the local package list from repositories
- **Search and inspect**: `apt search` finds packages, `apt show` gives details
- **Install packages**: `apt install -y` downloads and installs software with dependencies
- **Find binaries**: `which` shows the path, `whereis` shows binary, source, and docs
- **List installed**: `dpkg -l | grep "^ii"` counts all installed packages

In Kubernetes, you'll use these skills constantly -- installing `kubectl`, `helm`, `jq`, `curl`, and debugging tools on nodes. Many troubleshooting scenarios start with "install a tool to inspect the problem."

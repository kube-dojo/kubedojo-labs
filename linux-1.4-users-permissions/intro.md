## Users and Permissions

User management is fundamental to Linux security. Every process runs as a user, every file is owned by a user, and access control starts here.

### What You'll Learn

- Create and manage users and groups
- Understand password storage and account locking
- Configure fine-grained sudo access
- Set up shared project directories with correct ownership

### Before You Start

If you haven't read the theory yet, check out [Module 1.4: Users and Permissions](https://kube-dojo.github.io/linux/foundations/system-essentials/module-1.4-users-permissions/) on KubeDojo.

This lab takes about **30 minutes**. Let's go!

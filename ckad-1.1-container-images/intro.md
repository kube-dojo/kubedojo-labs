# Container Images

In this lab you will work with container image pull policies, tags, and troubleshoot common image-related errors.

Understanding how Kubernetes pulls and caches images is essential for CKAD tasks involving deployments and pod specifications.

**What you will learn:**
- Configure imagePullPolicy (Always, IfNotPresent, Never)
- Use specific image tags vs. latest
- Debug ImagePullBackOff errors

**Time:** 25 minutes

**Curriculum module:** [CKAD Part 1 - Container Images](https://kube-dojo.github.io/k8s/ckad/part1-design-build/module-1.1-container-images/)

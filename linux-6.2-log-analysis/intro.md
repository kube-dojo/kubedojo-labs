# Log Analysis Techniques

Logs are the primary source of truth when diagnosing issues. This lab teaches you to efficiently parse, search, and correlate log data using command-line tools.

## What You Will Learn

- Parsing Apache/Nginx access logs
- Extracting top IPs, error codes, and request patterns
- Combining grep, awk, sed, sort, and uniq for powerful analysis
- Correlating entries across different log sources

## Prerequisites

- Basic grep and pipe usage
- Understanding of HTTP status codes

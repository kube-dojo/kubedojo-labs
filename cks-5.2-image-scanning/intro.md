# Image Vulnerability Scanning

Scanning images for known CVEs is a critical supply chain security control. This lab covers trivy for comprehensive vulnerability detection.

## What You Will Learn

- Scan images for OS and library vulnerabilities
- Filter results by severity and fixability
- Scan all images running in a Kubernetes cluster

# Resource Requirements and Limits

In this lab you will configure CPU and memory requests and limits, create LimitRanges for default values, and set ResourceQuotas for namespace-level controls.

**What you will learn:**
- Set container resource requests and limits
- Create LimitRanges for defaults and constraints
- Apply ResourceQuotas to namespaces

**Time:** 30 minutes

**Curriculum module:** [CKAD Part 4 - Resources](https://kube-dojo.github.io/k8s/ckad/part4-environment/module-4.3-resources/)

# Container Logging

In this lab you will practice viewing, filtering, and managing container logs — a core CKAD skill for debugging applications.

**What you will learn:**
- kubectl logs with flags (--tail, --since, -c, -p)
- Multi-container pod log viewing
- Searching and saving log output

**Time:** 25 minutes

**Curriculum module:** [CKAD Part 3 - Container Logging](https://kube-dojo.github.io/k8s/ckad/part3-observability/module-3.2-logging/)

# Congratulations!

You have completed the Text Processing lab.

## What You Learned

- `awk` for field extraction, filtering, and arithmetic
- `sed` for search-and-replace and stream transformations
- Building powerful pipelines with sort, uniq, and xargs

## Key Takeaways

1. **awk** excels at columnar data — CSV, TSV, log files
2. **sed** is best for in-place edits and pattern-based transformations
3. **Pipelines** combine simple tools into powerful data processing workflows
4. **sort | uniq -c | sort -rn** is the frequency analysis pattern you will use constantly

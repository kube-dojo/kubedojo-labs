# AppArmor Profiles for Containers

AppArmor is a Linux kernel security module that restricts program capabilities. Kubernetes supports applying AppArmor profiles to containers.

## What You Will Learn

- List and load AppArmor profiles on nodes
- Apply profiles to pods using annotations or security context
- Create custom profiles for specific workloads
- Troubleshoot AppArmor-related pod failures

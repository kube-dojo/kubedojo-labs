# Congratulations!

You have completed the **AppArmor Profiles** lab.

## Key Takeaways

- AppArmor restricts container capabilities at the kernel level
- Profiles must be loaded on every node where pods run
- Use complain mode for debugging before enforcing profiles
- Custom profiles should allow only what the application needs

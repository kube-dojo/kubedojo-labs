# Kubernetes Documentation

In the CKA exam, you have access to the official Kubernetes documentation. But the fastest way to look up resource fields is often `kubectl explain` and `kubectl api-resources` right in the terminal.

In this lab you will:

- Use `kubectl explain` to explore API resource fields
- Use `kubectl api-resources` to discover resource types and short names
- Generate YAML manifests with `--dry-run=client -o yaml`

**What you will learn:**
- How to browse the Kubernetes API from the command line
- How to find short names and API groups for resources
- How to generate starter YAML without memorizing every field

**Time:** 20 minutes

**Curriculum module:** [CKA Part 0 - K8s Docs](https://kube-dojo.github.io/k8s/cka/part0-environment/module-0.4-k8s-docs/)

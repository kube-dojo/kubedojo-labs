## Your First Kubernetes Cluster

Welcome to your first Kubernetes lab! You have a real, single-node Kubernetes cluster running right here. Everything you type runs against a live cluster.

### What You'll Learn

- Inspect cluster information with `kubectl cluster-info` and `kubectl get nodes`
- Discover control plane components running in `kube-system`
- Create and verify a Kubernetes namespace

### Before You Start

If you haven't read the theory yet, check out [Module 1.1: Your First Cluster](https://kube-dojo.github.io/prerequisites/kubernetes-basics/module-1.1-first-cluster/) on KubeDojo.

This lab takes about **25 minutes**. Type the commands yourself — muscle memory is the goal.

Let's go!

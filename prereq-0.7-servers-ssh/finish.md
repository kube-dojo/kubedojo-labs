### What You Learned

- **SSH keys**: `ssh-keygen -t ed25519` generates a modern key pair for secure authentication
- **Listening ports**: `ss -tlnp` shows which services are running and on which ports
- **SSH config**: `/etc/ssh/sshd_config` controls how the SSH server behaves
- **File integrity**: `md5sum` computes checksums to verify files haven't been altered

These skills are fundamental for managing Kubernetes nodes. You'll use SSH to access nodes, check ports to debug service connectivity, and verify downloads when installing tools like `kubectl` and `helm`.

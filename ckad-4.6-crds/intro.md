# Custom Resource Definitions (CRDs)

In this lab you will create CRDs to extend the Kubernetes API with your own resource types, then create and manage instances of those resources.

**What you will learn:**
- Define Custom Resource Definitions
- Create custom resources
- Query custom resources with kubectl

**Time:** 30 minutes

**Curriculum module:** [CKAD Part 4 - CRDs](https://kube-dojo.github.io/k8s/ckad/part4-environment/module-4.6-crds/)

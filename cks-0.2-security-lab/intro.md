# Security Lab Environment

A well-configured lab environment is essential for practicing CKS exam scenarios. This lab guides you through setting up security tools, audit logging, and test workloads.

## What You Will Learn

- Install security scanning tools (trivy, kubesec)
- Configure basic audit logging
- Deploy intentionally vulnerable workloads for testing
- Document security baselines

## Prerequisites

- Kubernetes cluster (provided)
- Basic kubectl skills

> **Study guide**: [KubeDojo Security Lab](https://kube-dojo.github.io/k8s/cks/part0/module-0.2-security-lab/)

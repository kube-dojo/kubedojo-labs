# CIS Benchmarks with kube-bench

The CIS Kubernetes Benchmark provides prescriptive guidance for securing Kubernetes clusters. kube-bench automates these checks.

## What You Will Learn

- Run kube-bench against master and worker components
- Interpret CIS benchmark results
- Remediate common API server and kubelet failures

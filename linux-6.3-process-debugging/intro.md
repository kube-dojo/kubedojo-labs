# Process Debugging with strace, lsof, and ltrace

When a process misbehaves, you need to look inside it. Linux provides powerful tools for runtime debugging without modifying the application.

## What You Will Learn

- **strace** — trace system calls (file operations, network, memory)
- **lsof** — list all files opened by a process
- **ltrace** — trace library function calls

## Prerequisites

- Understanding of processes and PIDs
- Basic knowledge of system calls and libraries

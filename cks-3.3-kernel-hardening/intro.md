# Kernel-Level Hardening

Linux kernel parameters and capabilities directly affect container security. This lab covers sysctl configuration, capability management, and system-level hardening.

## What You Will Learn

- Configure kernel sysctl parameters for container security
- Drop unnecessary Linux capabilities from containers
- Apply system-level hardening best practices

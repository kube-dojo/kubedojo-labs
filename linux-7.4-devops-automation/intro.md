# DevOps Automation Scripts

Shell scripts are the glue of DevOps workflows. This lab covers three common automation patterns: configuration templating, API interaction, and CI/CD pipeline scripting.

## What You Will Learn

- Configuration templating with `envsubst`
- REST API calls with `curl` and JSON parsing with `jq`
- Simulating a build/test/deploy pipeline in bash

## Prerequisites

- Bash scripting fundamentals (variables, functions, conditionals)
- Basic understanding of HTTP and JSON

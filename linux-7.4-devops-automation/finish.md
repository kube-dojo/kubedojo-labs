## What You Learned

- Configuration templating with `envsubst`
- REST API interaction with `curl` and `jq`

## Key Takeaways

1. **envsubst** is how most containers do configuration — learn it well
2. **curl + jq** is the CLI equivalent of Postman for API debugging
3. **Pipeline scripts** should fail fast — use `set -e` and check return codes
4. These patterns are the foundation of real CI/CD systems (GitHub Actions, Jenkins, etc.)

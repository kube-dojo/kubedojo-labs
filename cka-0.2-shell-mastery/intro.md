# Shell Mastery for CKA

The CKA exam is a hands-on, timed exam. Every second counts. Shell efficiency can save you 10-15 minutes over the course of the exam.

In this lab you will practice:

- Bash shortcuts and aliases that speed up kubectl usage
- Output filtering with grep, awk, and jq
- Using tmux for multiple terminals (available in the exam)
- Timed scripting to build speed

**What you will learn:**
- Essential bash aliases for kubectl
- Text processing with grep, awk, and jq
- tmux session management
- Writing quick bash scripts under time pressure

**Time:** 30 minutes

**Curriculum module:** [CKA Part 0 - Shell Mastery](https://kube-dojo.github.io/k8s/cka/part0-environment/module-0.2-shell-mastery/)

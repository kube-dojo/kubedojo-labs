# ServiceAccounts

In this lab you will create ServiceAccounts and assign them to pods. ServiceAccounts provide an identity for processes running in pods.

**What you will learn:**
- Create ServiceAccounts
- Assign ServiceAccounts to pods
- Control API token auto-mounting

**Time:** 25 minutes

**Curriculum module:** [CKAD Part 4 - ServiceAccounts](https://kube-dojo.github.io/k8s/ckad/part4-environment/module-4.5-serviceaccounts/)

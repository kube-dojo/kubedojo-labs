# Ingress

In this lab you will create Ingress resources for HTTP routing. Ingress provides path-based and host-based routing to Services.

**What you will learn:**
- Set up an Ingress controller
- Create path-based Ingress rules
- Create host-based Ingress rules

**Time:** 35 minutes

**Curriculum module:** [CKAD Part 5 - Ingress](https://kube-dojo.github.io/k8s/ckad/part5-networking/module-5.2-ingress/)

# Compromised Container Investigation

When a container is compromised, you need to investigate, collect evidence, contain the threat, and remediate. This lab simulates a security incident.

## What You Will Learn

- Identify indicators of compromise in containers
- Collect forensic evidence before destroying containers
- Contain incidents using NetworkPolicies and pod deletion
- Document findings for incident reports

> **Study guide**: [KubeDojo Container Investigation](https://kube-dojo.github.io/k8s/cks/part6/module-6.3-container-investigation/)

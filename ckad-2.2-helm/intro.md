# Helm Package Manager

In this lab you will use Helm to install, configure, upgrade, and roll back applications. Helm is part of the CKAD exam curriculum.

**What you will learn:**
- Add repositories and search for charts
- Install releases with custom values
- Upgrade and rollback releases

**Time:** 30 minutes

**Curriculum module:** [CKAD Part 2 - Helm](https://kube-dojo.github.io/k8s/ckad/part2-deployment/module-2.2-helm/)

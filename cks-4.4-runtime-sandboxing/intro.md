# Runtime Sandboxing

Container sandboxing provides additional isolation between containers and the host kernel. This lab covers RuntimeClass, gVisor, and Kata Containers concepts.

## What You Will Learn

- Understand the RuntimeClass API
- Create RuntimeClass resources for different sandbox levels
- Compare isolation tradeoffs between runc, gVisor, and Kata Containers

> **Study guide**: [KubeDojo Runtime Sandboxing](https://kube-dojo.github.io/k8s/cks/part4/module-4.4-runtime-sandboxing/)

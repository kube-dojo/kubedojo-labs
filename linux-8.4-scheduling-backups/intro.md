# Scheduling and Backup Strategies

Automated scheduling and backup rotation are fundamental sysadmin tasks. This lab covers cron for recurring tasks, `at` for one-time jobs, and implementing a backup rotation strategy.

## What You Will Learn

- Creating and managing cron jobs
- Scheduling one-time tasks with `at`
- Implementing backup rotation to manage disk space

## Prerequisites

- Basic bash scripting
- Understanding of cron syntax

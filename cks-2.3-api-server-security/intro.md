# API Server Security

The API server is the gateway to your cluster. Hardening it is critical for CKS (Cluster Hardening domain, 15%).

## What You Will Learn

- Review and strengthen authentication mechanisms
- Configure proper authorization modes
- Enable security-relevant admission controllers

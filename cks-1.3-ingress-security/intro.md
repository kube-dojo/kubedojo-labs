# Ingress Security & TLS

Securing Ingress is critical for protecting web-facing applications. This lab covers TLS termination, certificate management, and security headers.

## What You Will Learn

- Generate TLS certificates and store them as Kubernetes Secrets
- Configure Ingress resources with TLS termination
- Add security headers and HTTPS redirect

> **Study guide**: [KubeDojo Ingress Security](https://kube-dojo.github.io/k8s/cks/part1/module-1.3-ingress-security/)

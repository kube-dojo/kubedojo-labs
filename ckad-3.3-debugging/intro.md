# Debugging in Kubernetes

In this lab you will practice debugging common Kubernetes issues — CrashLoopBackOff, Pending pods, and container networking problems. These debugging skills are essential for the CKAD exam.

**What you will learn:**
- Diagnose CrashLoopBackOff with logs and describe
- Fix Pending pods caused by resource constraints
- Use kubectl exec and ephemeral debug containers

**Time:** 35 minutes

**Curriculum module:** [CKAD Part 3 - Debugging](https://kube-dojo.github.io/k8s/ckad/part3-observability/module-3.3-debugging/)

## CLI Power User

Good terminal users know the basics. Great terminal users work 10x faster with shortcuts, pipes, and text processing.

### What You'll Learn

- Use command history, shortcuts, and aliases to save keystrokes
- Chain commands with pipes and redirect output to files
- Process text with `grep`, `sed`, `awk`, `cut`, `sort`, and `uniq`
- Boost productivity with `xargs`, `find -exec`, and `watch`

### Before You Start

If you haven't read the theory yet, check out [Module 0.1: CLI Power User](https://kube-dojo.github.io/linux/foundations/everyday-use/module-0.1-cli-power-user/) on KubeDojo.

This lab takes about **30 minutes**. Type every command yourself — muscle memory matters.

Let's go!

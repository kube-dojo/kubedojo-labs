# CRDs and Operators

Custom Resource Definitions (CRDs) extend the Kubernetes API with new resource types. Operators use CRDs combined with controllers to automate application management.

In this lab you will:

- Explore existing CRDs in the cluster
- Create a custom CRD defining a new resource type
- Instantiate a custom resource from your CRD

**What you will learn:**
- How to list and inspect CRDs
- How to write a CRD manifest with schema validation
- How to create instances of custom resources

**Time:** 40 minutes

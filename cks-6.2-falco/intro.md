# Falco Runtime Security

Falco is a cloud-native runtime security tool that detects anomalous activity in containers, hosts, and Kubernetes. It is a CNCF graduated project.

## What You Will Learn

- Install and configure Falco for container monitoring
- Write custom Falco rules for Kubernetes security
- Detect and respond to runtime security events

> **Study guide**: [KubeDojo Falco](https://kube-dojo.github.io/k8s/cks/part6/module-6.2-falco/)

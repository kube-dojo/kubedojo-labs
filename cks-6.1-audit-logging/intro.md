# Kubernetes Audit Logging

Audit logging records all API requests for security monitoring and forensics (Runtime Security domain, 20% CKS weight).

## What You Will Learn

- Create audit policies with multiple rules and levels
- Enable audit logging on the API server
- Analyze audit logs to detect security events
- Build audit alerting rules

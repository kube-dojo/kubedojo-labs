# ConfigMaps

In this lab you will create ConfigMaps and use them to inject configuration into pods via environment variables and volume mounts.

**What you will learn:**
- Create ConfigMaps from literals, files, and directories
- Inject ConfigMap data as environment variables
- Mount ConfigMaps as files in pods

**Time:** 30 minutes

**Curriculum module:** [CKAD Part 4 - ConfigMaps](https://kube-dojo.github.io/k8s/ckad/part4-environment/module-4.1-configmaps/)

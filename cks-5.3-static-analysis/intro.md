# Static Analysis of Kubernetes Manifests

Static analysis catches security issues in YAML manifests before deployment. This lab covers kubesec, manual review patterns, and security checklists.

## What You Will Learn

- Score manifests with kubesec
- Identify security anti-patterns in Kubernetes YAML
- Create a reusable security review checklist

> **Study guide**: [KubeDojo Static Analysis](https://kube-dojo.github.io/k8s/cks/part5/module-5.3-static-analysis/)

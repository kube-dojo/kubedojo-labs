# Cluster Setup Verification

In this lab you will verify that a Kubernetes cluster is healthy, explore its core components, and configure a custom kubectl context.

These are foundational skills for the CKA exam. Before solving any exam task, you should always confirm your cluster is operational and that your kubectl is pointed at the correct context.

**What you will learn:**
- How to check node and component status
- How to identify control plane pods in kube-system
- How to create and switch kubectl contexts

**Time:** 30 minutes

**Curriculum module:** [CKA Part 0 - Cluster Setup](https://kube-dojo.github.io/k8s/cka/part0-environment/module-0.1-cluster-setup/)

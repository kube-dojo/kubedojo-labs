# Deployment Strategies

In this lab you will implement different deployment strategies. Understanding when to use each strategy and how to configure them is tested on the CKAD.

**What you will learn:**
- Configure maxSurge and maxUnavailable for rolling updates
- Use Recreate strategy for breaking changes
- Implement blue-green deployments with service switching

**Time:** 35 minutes

**Curriculum module:** [CKAD Part 2 - Deployment Strategies](https://kube-dojo.github.io/k8s/ckad/part2-deployment/module-2.4-deployment-strategies/)

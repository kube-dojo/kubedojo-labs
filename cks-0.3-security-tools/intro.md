# Essential Security Tools

Security tools are a critical part of the CKS exam. You need to be proficient with kube-bench for CIS benchmarks, trivy for image scanning, and kubesec for manifest analysis.

## What You Will Learn

- Run CIS Kubernetes Benchmark with kube-bench
- Scan images for CVEs with trivy
- Analyze pod security with kubesec

> **Study guide**: [KubeDojo Security Tools](https://kube-dojo.github.io/k8s/cks/part0/module-0.3-security-tools/)

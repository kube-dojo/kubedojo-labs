## Lab Complete!

You've explored your first Kubernetes cluster:

- **Cluster info**: `kubectl cluster-info` and `kubectl get nodes` — see what's running
- **Control plane**: `kubectl get pods -n kube-system` — discover the components that make Kubernetes work
- **Namespaces**: `kubectl create namespace` — organize resources into logical groups

Every Kubernetes journey starts here. These commands will become second nature.

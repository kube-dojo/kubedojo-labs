### What You Learned

- **Copy files**: `cp source destination` -- duplicates a file
- **Move/rename files**: `mv source destination` -- moves or renames
- **Delete files**: `rm file` -- permanently removes a file
- **Delete directories**: `rm -r dir` -- recursively removes a directory and its contents
- **Change permissions**: `chmod 600 file` (numeric) or `chmod u+x file` (symbolic)
- **Absolute paths**: Start from `/`, work from anywhere
- **Relative paths**: Start from your current directory

These are the building blocks of file management. You'll use them constantly as you work with Kubernetes configs, scripts, and manifests.

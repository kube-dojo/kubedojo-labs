## Files and Directories

Welcome to the Files and Directories lab from KubeDojo! You'll practice the commands that let you organize, protect, and navigate your files.

### What You'll Learn

- Copy and move files with `cp` and `mv`
- Delete files and directories with `rm` and `rm -r`
- Change file permissions with `chmod`
- Navigate using absolute and relative paths

### Before You Start

If you haven't read the theory yet, check out [Module 0.4: Files and Directories](https://kube-dojo.github.io/prerequisites/zero-to-terminal/module-0.4-files-directories/) on KubeDojo.

This lab takes about **20 minutes**. Type the commands yourself -- muscle memory is the goal.

Let's go!

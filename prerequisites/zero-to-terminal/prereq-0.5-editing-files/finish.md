### What You Learned

- **Multi-line files**: `echo -e` with `\n`, heredocs (`cat << EOF`), and multiple redirections
- **Append operator**: `>>` adds to the end of a file without overwriting
- **Stream editing**: `sed -i 's/old/new/g'` finds and replaces text in place

These skills are essential for working with Kubernetes manifests, environment files, and configuration management. You'll use heredocs constantly when writing YAML for pods, deployments, and services.

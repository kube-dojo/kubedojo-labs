## First Terminal Commands

Welcome to your first hands-on lab! This is a real Linux terminal — everything you type runs on a real machine.

### What You'll Learn

- Navigate the filesystem with `pwd`, `ls`, and `cd`
- Create files and directories with `touch`, `mkdir`, and `echo`
- Read file contents with `cat`, `head`, and `grep`

### Before You Start

If you haven't read the theory yet, check out [Module 0.3: First Commands](https://kube-dojo.github.io/prerequisites/zero-to-terminal/module-0.3-first-commands/) on KubeDojo.

This lab takes about **20 minutes**. Type the commands yourself — muscle memory is the goal.

Let's go!

# Practical Shell Scripts

Theory becomes powerful when applied. In this lab you will build three scripts that solve real operational tasks: automated backups, system health checks, and HTML report generation.

## What You Will Learn

- Creating date-stamped backup archives
- Building multi-check health monitoring scripts
- Generating HTML reports from system data

## Prerequisites

- Bash variables, loops, functions, and conditionals
- Basic understanding of `tar`, `df`, `free`

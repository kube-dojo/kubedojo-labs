# Congratulations!

You have completed the Kustomize lab. You now know how to:

- Create a base kustomization with resources
- Add overlays with namePrefix and replica patches
- Preview with `kubectl kustomize` and apply with `kubectl apply -k`

Kustomize is built into kubectl and is a key tool for the CKA exam, especially for managing environment-specific configurations.

# Host & Network Security

System hardening extends beyond Kubernetes to the host OS and network layer. This lab covers minimizing the host attack surface and securing network access.

## What You Will Learn

- Audit and reduce unnecessary services on K8s nodes
- Prevent pods from accessing host namespaces and networks
- Configure host firewall rules for node protection

> **Study guide**: [KubeDojo Network Security](https://kube-dojo.github.io/k8s/cks/part3/module-3.4-network-security/)

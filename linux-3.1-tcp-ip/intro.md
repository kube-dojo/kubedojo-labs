# TCP/IP Networking Fundamentals

In this lab you will explore the core building blocks of TCP/IP networking on Linux. You will work with IP addresses and CIDR subnets, inspect and understand routing tables, observe TCP connections in real time, and map ports to well-known services.

These skills are essential for any Linux administrator or cloud-native engineer — every Kubernetes pod, service, and ingress ultimately relies on TCP/IP.

## What You Will Learn

- How to list and interpret IP addresses with `ip addr`
- How Linux routing tables determine where packets go
- How to observe TCP connection states with `ss`
- How `/etc/services` maps port numbers to protocol names

## Prerequisites

- Basic Linux command-line familiarity
- Understanding of what an IP address is

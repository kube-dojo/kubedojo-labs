# kubeadm

kubeadm is the official tool for creating and managing Kubernetes clusters. Understanding kubeadm is essential for the CKA exam, which includes cluster lifecycle tasks like upgrades and certificate management.

In this lab you will:

- Inspect the cluster configuration created by kubeadm
- Check certificate expiration dates
- Run an upgrade plan to see available versions

**What you will learn:**
- How kubeadm stores cluster configuration
- How to check and manage TLS certificates
- How to plan a cluster upgrade

**Time:** 40 minutes

**Curriculum module:** [CKA Part 1 - kubeadm](https://kube-dojo.github.io/k8s/cka/part1-cluster-architecture/module-1.7-kubeadm/)

# Dashboard & GUI Security

The Kubernetes Dashboard is a common attack vector when misconfigured. This lab covers secure deployment, RBAC restrictions, and access auditing.

## What You Will Learn

- Deploy the dashboard with security-hardened settings
- Create least-privilege RBAC for dashboard users
- Audit dashboard access and permissions

> **Study guide**: [KubeDojo GUI Security](https://kube-dojo.github.io/k8s/cks/part1/module-1.5-gui-security/)

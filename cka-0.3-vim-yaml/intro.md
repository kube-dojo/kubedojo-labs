# Vim and YAML

The CKA exam requires you to create and edit YAML manifests quickly. Vim is the default editor and YAML is the language of Kubernetes.

In this lab you will:

- Learn essential vim commands for navigating and editing files
- Understand YAML structure and fix common indentation errors
- Configure vim for comfortable YAML editing

**What you will learn:**
- Vim insert mode, navigation, search, save/quit
- YAML indentation rules, lists, maps, and multi-line strings
- Vim settings for YAML: expandtab, tabstop, autoindent

**Time:** 25 minutes

**Curriculum module:** [CKA Part 0 - Vim and YAML](https://kube-dojo.github.io/k8s/cka/part0-environment/module-0.3-vim-yaml/)

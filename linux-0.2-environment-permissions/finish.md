## Lab Complete!

You've mastered the environment and permissions fundamentals:

- **Environment variables**: `export`, `PATH`, persisting in `~/.bashrc`
- **File permissions**: `chmod`, `chown`, `umask` — the classic Unix permission model
- **Special bits**: SUID, SGID, sticky bit — elevated and shared permissions
- **ACLs**: `setfacl`, `getfacl` — fine-grained access beyond owner/group/other

These concepts are critical for Kubernetes security contexts, container permissions, and cloud IAM.

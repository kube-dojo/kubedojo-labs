## Environment and Permissions

Environment variables control how programs behave. File permissions control who can access what. Together they form the security foundation of every Linux system.

### What You'll Learn

- Set and persist environment variables with `export` and `~/.bashrc`
- Understand and modify file permissions with `chmod`, `chown`, and `umask`
- Work with special permission bits: SUID, SGID, sticky bit
- Use POSIX ACLs for fine-grained access control

# Kubernetes Version Upgrades

Running outdated Kubernetes versions exposes your cluster to known CVEs. Regular upgrades are a security requirement.

## What You Will Learn

- Check current version and available upgrades
- Plan an upgrade path following Kubernetes version policy
- Verify cluster health after upgrade

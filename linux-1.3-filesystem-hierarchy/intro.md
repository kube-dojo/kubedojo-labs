## Filesystem Hierarchy

Every Linux system follows the **Filesystem Hierarchy Standard (FHS)**. Knowing where things live helps you navigate any system — from a bare metal server to a Kubernetes node.

### What You'll Learn

- Understand the purpose of key directories: `/etc`, `/var`, `/tmp`, `/usr`, `/opt`
- Find files efficiently with `find`, `locate`, `which`, `whereis`
- Inspect disk usage and mount points with `df`, `mount`, `lsblk`

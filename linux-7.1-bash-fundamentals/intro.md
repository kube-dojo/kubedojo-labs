# Bash Scripting Fundamentals

Bash scripting is a core skill for any Linux administrator or DevOps engineer. Scripts automate repetitive tasks, enforce consistency, and enable complex workflows.

## What You Will Learn

- Variables, string operations, and arithmetic
- Conditionals with `if`, `test`, and `[[ ]]`
- Loops: `for`, `while`, `until`
- Functions for code reuse
- Error handling with `set -e`, `trap`, and exit codes

## Prerequisites

- Basic Linux command line proficiency
- Ability to create and edit files

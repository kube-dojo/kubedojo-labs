## Lab Complete!

You've gained solid process management skills:

- **Process listing**: `ps aux`, `pstree` — see what's running
- **Signals & jobs**: `kill`, `bg`, `fg`, `nohup` — control process lifecycle
- **Resource monitoring**: `free`, `df`, `du` — track memory and disk
- **Limits**: `ulimit`, `/proc/PID/limits` — understand resource boundaries

These skills directly apply to Kubernetes: pods are processes, resource limits map to cgroup constraints, and signals control container shutdown.

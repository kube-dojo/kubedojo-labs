# Helm

Helm is the package manager for Kubernetes. It uses **charts** (packaged applications) to deploy and manage complex applications.

In this lab you will:

- Install Helm from the official script
- Add chart repositories and search for charts
- Install a chart as a release
- Upgrade a release and rollback to a previous revision

**What you will learn:**
- Helm installation and version verification
- Repository management with `helm repo`
- Release lifecycle: install, upgrade, rollback, history
- Using `--set` to override chart values

**Time:** 40 minutes

**Curriculum module:** [CKA Part 1 - Helm](https://kube-dojo.github.io/k8s/cka/part1-cluster-architecture/module-1.3-helm/)

## Kernel Architecture

The Linux kernel is the foundation everything runs on. Understanding it helps you debug low-level issues, tune performance, and understand how containers actually work.

### What You'll Learn

- Inspect kernel version and loaded modules with `uname`, `lsmod`, `modinfo`
- Explore the `/proc` virtual filesystem
- Read and set kernel parameters with `sysctl`
- Analyze kernel logs with `dmesg`

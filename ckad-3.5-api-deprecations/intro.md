# API Deprecations

In this lab you will learn how to identify and handle API deprecations. Kubernetes regularly deprecates and removes API versions — knowing how to migrate manifests is a CKAD exam topic.

**What you will learn:**
- List available API versions and resources
- Identify deprecated API versions in manifests
- Migrate manifests to current API versions

**Time:** 25 minutes

**Curriculum module:** [CKAD Part 3 - API Deprecations](https://kube-dojo.github.io/k8s/ckad/part3-observability/module-3.5-api-deprecations/)

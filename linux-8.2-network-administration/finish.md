# Congratulations!

You have completed the Network Administration lab.

## What You Learned

- Creating and configuring virtual network interfaces
- Managing the routing table with `ip route`
- Configuring firewall rules with `iptables`

## Key Takeaways

1. **ip** is the modern replacement for `ifconfig` and `route`
2. **Static routes** are essential for multi-subnet environments
3. **iptables rules are ordered** — first match wins
4. **Always allow loopback and established connections** before restricting traffic

# CKS Exam Overview & Strategy

The **Certified Kubernetes Security Specialist (CKS)** exam tests your ability to secure Kubernetes clusters and workloads. It is a hands-on, performance-based exam lasting **2 hours**.

## What You Will Learn

- The CKS exam domains and their weights
- How to quickly assess cluster security posture
- Time management strategies for the exam

## CKS Domains

| Domain | Weight |
|--------|--------|
| Cluster Setup | 10% |
| Cluster Hardening | 15% |
| System Hardening | 15% |
| Minimize Microservice Vulnerabilities | 20% |
| Supply Chain Security | 20% |
| Monitoring, Logging and Runtime Security | 20% |

## Prerequisites

- CKA certification (required)
- Familiarity with Linux security primitives

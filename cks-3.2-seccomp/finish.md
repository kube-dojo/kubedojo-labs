# Congratulations!

You have completed the **Seccomp Profiles** lab.

## Key Takeaways

- RuntimeDefault blocks ~50 dangerous syscalls with no configuration
- Custom Localhost profiles provide maximum restriction
- Use SCMP_ACT_LOG audit mode to discover required syscalls
- Profiles must exist on every node where pods can be scheduled
